# Installation
```pip
pip3 install -U nwpy
```
# Basic Usage
Simply run ```nwpy```, It will look for your default interface and start scanning.
```bash
$ nwpy --help
usage: nwpy [-h] [-i I] [-l]

Network scanner

optional arguments:
  -h, --help  show this help message and exit
  -i I        interface
  -l          List available interfaces
```
